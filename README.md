# Unreal Commandlet Tool

This repository contains experimental and research code developed in spare time to explore custom Unreal Engine commandlets and automation workflows.

The project is intended for learning, testing, and reference purposes.  
It is not a production-ready solution and may change frequently.

Use at your own discretion.

# Build Tools

This folder contains local automation scripts and batch files used for testing and running custom Unreal Engine commandlets.

It is mainly intended for development, experimentation, and build automation during spare time research.

Log files generated by these tools are stored in the `ToolRunner.log` file.
